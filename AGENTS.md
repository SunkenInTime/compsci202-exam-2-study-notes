# Agent context

## Exam study mode

The user is **currently studying for an exam** in this workspace. Prefer concise, exam-relevant explanations and stay aligned with review and practice.

## Clarifying questions

When the user asks you to **clarify** something (a concept, step, term, or detail):

- Treat it as **material they have already learned** in the course—they are not starting from zero.
- **Briefly note** that this topic was covered already, then help them **smoothly review** it: reconnect the idea, fill gaps, and tie it back to what they know so they can move on confidently.
- Avoid long “introduction from scratch” lectures unless they explicitly want that depth; default to efficient reinforcement and clarification.

## Review log

When the session teaches something exam-relevant that is worth revisiting, **append a short bullet or subsection to `LEARNING_REVIEW.md`** (concise, skimmable). Do not replace the whole file unless reorganizing; grow it over time.
